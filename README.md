# Actions-OpenWrt-Newifi3-d2

**基于 [Actions-OpenWrt](https://github.com/P3TERX/Actions-OpenWrt) 项目云编译的Openwrt固件**

固件适用型号: **Newifi3-d2**

- Theme: Argon(默认)、Bootstrap
- IP: 192.168.10.1
- Language: en; zh-cn
- 用户名: root
- 密码: password
